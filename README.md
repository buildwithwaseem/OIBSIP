## 🌸 Task 1: Iris Flower Classification
The goal of this project is to train a machine learning model that can learn from the measurements of iris flower species and classify them into three categories: **Setosa, Versicolor, and Virginica**.

### 🛠️ Tools & Technologies
- **Language:** Python
- **Environment:** Google Colab / Ubuntu (WSL)
- **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

### 📊 Dataset Details
The dataset contains 150 records with 5 key features:
1. Sepal Length (cm)
2. Sepal Width (cm)
3. Petal Length (cm)
4. Petal Width (cm)
5. Species (Target)

### 🚀 Implementation Steps
1. **Data Loading:** Read the `Iris.csv` file using Pandas.
2. **EDA (Exploratory Data Analysis):** Visualized the relationship between features using Pairplots and Heatmaps.
3. **Preprocessing:** Dropped unnecessary columns and split the data into training (80%) and testing (20%) sets.
4. **Model Training:** Implemented **Logistic Regression** for classification.
5. **Evaluation:** Achieved high accuracy on the test set.

- **Model Accuracy:** ~100% (or your specific accuracy)
- **Key Insight:** Petal measurements are the strongest indicators for species differentiation.
